# express_blog

这是一个使用 express 做的 blog demo

目前已完成的功能:

1. 用户注册,登录,注销
2. 使用 markdown 发布文章

待完成功能:

1. 评论功能
2. 权限分级
3. 文章分页
4. 分类，标签
...

使用技术:

1. express 框架
2. bootstrap 前端框架
3. mongodb 数据库

运行这个项目:

0. 前提，需要安装好 `node.js` 和 `mongodb`
1. 首先 clone 到本地
2. 进入项目目录执行 `npm install`
3. 运行 `npm start` 启动项目
4. 在浏览器中访问 `localhost:3000`
